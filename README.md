#Steam Summer 2015 Monster Minigame Number Formatter

Truncates numbers down to 3 digits.

For example:

| Before  | After |
| -------:| -----:|
|       1 |     1 |
|      21 |    21 |
|     321 |   321 |
|   4,321 | 4.32k |
|  64,321 | 54.3k |
|    654K |  654k |
|  7,654K | 7.65M |
| 87,654K | 87.6M |
|    987M |  987M |
